# Category Module

Browse content categories available on the platform.

## When to Use

When you need to see what content categories are available for organizing or filtering content.

## Subcommands

| Subcommand | Description |
|------------|-------------|
| `list` | List all categories |

## Usage

```bash
python {baseDir}/scripts/category.py list
```

## Options

| Option | Description |
|--------|-------------|
| `--params JSON` | Extra filter params |
